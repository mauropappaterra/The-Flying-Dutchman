The Flying Dutchman

This is our submission project for User Interface Programming I at Uppsala University

For this task we had to create an interface for a fictitious bar, but we were not allow to use any web framework or any other tool other than HTML, CSS and JavaScript (with JQuery). To complete this task, we had to implement a dummy database and create fully functional ordering system implementing interface design principles. Moreover, this website is fully responsive and implements undo/redo and drag and drop (try dragging the pirate flag to the ship).

Click on the green tab to get some passwords and try out our implementation. Scroll down for link to the repo on Github.
